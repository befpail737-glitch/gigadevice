## ⚠️ 常见问题解决

### 问题1: 地址验证失败

**解决方案**:
```
1. 确认地址格式正确
2. 检查邮编是否准确
3. 尝试添加 Suite/Room 号码
4. 联系Google支持: https://support.google.com/business/
```

### 问题2: 类别选择受限

**解决方案**:
```
1. 选择最接近的主要类别
2. 在描述中详细说明服务
3. 通过属性补充服务类型
4. 定期更新产品列表
```

### 问题3: 多地点管理

**解决方案**:
```
1. 使用Google Business Profile Manager
2. 为每个地点创建独立档案
3. 统一品牌信息，本地化内容
4. 使用批量管理工具
```
